
To bring up the server...
$ rackup --port=4567

In the browser go to
http://localhost:4567

To run the tests
$ rake test

Commands I ran to build the infrastructure

$ rvm get stable
$ rvm install ruby-2.4.0
$ gem install --no-rdoc --no-ri sinatra
$ gem install --no-rdoc --no-ri install bundler
$ bundle install


Build the Ubuntu server
$ curl -sSL https://get.docker.com/ | sh
$ sudo usermod -aG jrbjagger docker
$ sudo curl -o /usr/local/bin/docker-compose -L "https://github.com/docker/compose/releases/download/1.15.0/docker-compose-$(uname -s)-$(uname -m)"
$ sudo chmod +x /usr/local/bin/docker-compose
$ git clone https://github.com/JonJagger/chennai-students-server.git
$ cd chennai-students-server
$ ./up.sh
